# Security Implementation Guide

## Overview

The Quest Analytics RAG Assistant now includes comprehensive security enhancements:

- **API Authentication**: API key-based authentication for all endpoints
- **Input Sanitization**: Protection against XSS, SQL injection, and malware
- **Rate Limiting**: Intelligent throttling to prevent abuse
- **Security Headers**: Industry-standard security headers
- **Threat Detection**: Pattern-matching and automated blocking

## Quick Setup

### 1. Generate API Keys

```bash
# Generate user API key
python scripts/security_manager.py generate-key --type user

# Generate admin API key  
python scripts/security_manager.py generate-key --type admin
```

### 2. Configure Security Settings

```bash
# Enable authentication and set rate limits
python scripts/security_manager.py configure --enable-auth --rate-limit 100

# Test security components
python scripts/security_manager.py test-security
```

### 3. Environment Configuration

Add to your `.env` file:

```bash
# Security Configuration
SECURITY_ENABLED=true
ENVIRONMENT=production

# API Keys (generated by security_manager.py)
RAG_API_KEY=your_user_api_key_here
RAG_ADMIN_API_KEY=your_admin_api_key_here
```

## API Authentication

### Using API Keys

**Header Method:**
```bash
curl -H "X-API-Key: your_api_key" http://localhost:7860/api/chat
```

**Bearer Token Method:**
```bash
curl -H "Authorization: Bearer your_api_key" http://localhost:3000/api/health
```

## Security Features

### Input Sanitization

- **XSS Protection**: Removes `<script>`, `javascript:`, `onclick` patterns
- **SQL Injection**: Blocks `'; DROP TABLE`, `UNION SELECT` patterns  
- **File Validation**: Type checking, size limits, malware detection
- **Unicode Normalization**: Prevents encoding-based attacks

### Rate Limiting

```yaml
# Default Limits
global_limits:
  requests_per_minute: 60
  requests_per_hour: 1000

# Endpoint-specific limits  
endpoint_limits:
  "/api/chat": {requests_per_minute: 20}
  "/api/upload": {requests_per_minute: 5}
```

### Security Headers

All responses include:

- `X-Content-Type-Options: nosniff`
- `X-Frame-Options: DENY`
- `X-XSS-Protection: 1; mode=block`
- `Strict-Transport-Security: max-age=31536000`
- `Content-Security-Policy: default-src 'self'`

## Integration Examples

### Gradio App Integration

```python
from rag_pipeline.security import SecurityMiddleware

# Initialize security
security_middleware = SecurityMiddleware()

# In your function:
if security_middleware.auth.enabled:
    # Rate limiting
    is_allowed, limit_info = security_middleware.rate_limiter.check_rate_limit(
        client_ip, "/api/chat"
    )
    
    # Input sanitization
    sanitized_query, is_safe = security_middleware.sanitizer.sanitize_query(query)
```

### FastAPI Integration

```python
from rag_pipeline.security import SecurityMiddleware

security = SecurityMiddleware()

@app.middleware("http")
async def security_middleware(request: Request, call_next):
    # Apply rate limiting, headers, etc.
    return await security.process_request(request, call_next)
```

## Environment-Specific Settings

### Development
- Authentication: Disabled by default
- Rate limiting: Relaxed limits
- Detailed error messages

### Production  
- Authentication: Required
- Rate limiting: Strict limits
- Minimal error exposure
- Full security headers

### Configuration Override

```yaml
# configs/security_config.yaml
environments:
  development:
    authentication.enabled: false
    rate_limiting.enabled: false
    
  production:
    authentication.enabled: true
    rate_limiting.enabled: true
```

## Security Best Practices

1. **Rotate API Keys Regularly**: Generate new keys monthly
2. **Monitor Rate Limits**: Check for unusual traffic patterns
3. **Review Logs**: Monitor blocked requests and security events
4. **Update Dependencies**: Keep security dependencies current
5. **Backup Configuration**: Store security config in version control

## Troubleshooting

### Common Issues

**Authentication Failed:**
```bash
# Check if API key is set
echo $RAG_API_KEY

# Verify key format (should be 64 hex characters)
python scripts/security_manager.py test-security
```

**Rate Limit Exceeded:**
```bash
# Check current limits
python scripts/security_manager.py monitor

# Increase limits if needed
python scripts/security_manager.py configure --rate-limit 200
```

**Input Validation Errors:**
```bash
# Test sanitization
python scripts/security_manager.py test-security

# Check blocked patterns in configs/security_config.yaml
```
